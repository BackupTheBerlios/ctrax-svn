1) Install dependencies in unzipped, non-egg form:
$ easy_install.exe --always-unzip Ctrax
[http://www.py2exe.org/index.cgi/ExeWithEggs]
Remove installed Ctrax (C:\Python25\Lib\site-packages\Ctrax...) to avoid confusion (nec?)

2) Copy Ctrax subdirectories into py2exe directory. 

***
cp -r ../kcluster2d/* .
cp -r ../hungarian/* .
cp -r ../houghcircles/* .
cp -r ../Ctrax/* .
***

3) Copy motmot code into py2exe directory.

***
cp /c/Python25/Lib/site-packages/motmot/imops/imops.pyd .
cp /c/Python25/Lib/site-packages/motmot/ufmf/ufmf.py .
cp /c/Python25/Lib/site-packages/motmot/wxglvideo/wxglvideo.py .
cp /c/Python25/Lib/site-packages/motmot/wxglvideo/simple_overlay.py .
cp /c/Python25/Lib/site-packages/motmot/wxvalidatedtext/wxvalidatedtext.py .
cp /c/Python25/Lib/site-packages/motmot/wxvideo/wxvideo.py .
cp /c/Python25/Lib/site-packages/pygarrayimage/arrayimage.py .
***

4) Remove pkg_resources from motmot. 

***
python removepkgresources.py
***

Note: If the unused import of pkg_resources in simple_overlay were removed, this step may not be necessary. 

5) Replace import statements for motmot libraries
python replacemotmotimports.py

* Note that 4 and 5 must occur after 3 *

6) Compare the Ctrax/setup.py and py2exe/setup_py2exe.py scripts -- see if there are any changes to setup.py that need to manually be added to the old setup_py2exe.py script. Update version numbers in setup_py2exe.py and setup.nsi. 

7) Change scipy.interpolate.polyint to import factorial from scipy.misc.common:
Edit C:/Python25/Lib/site-packages/scipy/interpolate/polyint.py
change the line:
from scipy import factorial
to 
from scipy.misc.common import factorial

8) Try to build
$ python setup_py2exe.py build -c mingw32

9) Try it out:
$ python Ctrax-script.py

10) Try to make py2exe
$ python setup_py2exe.py build -c mingw32 py2exe

11) Try it out:
$ cd dist
$ ./Ctrax-script.exe

12) Make the Windows installer using NSIS:
Compile the script setup.nsi in the py2exe directory. (Start > NSIS, Compile NSI Scripts, File > Load > setup.nsi)

13) This makes a file Ctrax-(ver).exe in the current directory that can be distributed. Try it out by double-clicking and installing.
